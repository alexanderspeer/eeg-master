# EEG trainer

Browser drill: you get a fake 6 s EEG trace (256 Hz, one channel). Pick which rhythm band dominates (delta through gamma) and whether a special pattern is present (seizure types, spindles, K-complex, triphasic waves, burst suppression, or none). Ten rounds per session; feedback after each submit.

Data is synthesized in `eeg.js`, not from patients. For studying pattern recognition only, not diagnosis.

Open `index.html` in a browser. `guide.html` is the in-app reference.
